Security Model

Implemented

Event Flow Simulator

This simulates a denied process open/exec event going through the OmniKernel control path.

1. License
Validate signed license and node identity.
2. eBPF LSM
Attach file_open and bprm_check_security hooks.
3. Ring Buffer
Read security event from kernel ring buffer.
4. Signed Alert
Send HMAC signed payload to mesh hub.
5. Policy Runtime
Apply Wasmtime sandbox policy envelope.
Ready. Click Run Demo to simulate one security event path.
